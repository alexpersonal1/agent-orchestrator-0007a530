# Twister Serviços - Website de Logística Esportiva

## 🏆 Sobre o Projeto

Website profissional desenvolvido para a **Twister Serviços**, empresa especializada em logística esportiva com 8 anos de experiência no mercado baiano e brasileiro.

## ✨ Funcionalidades

- **Design Responsivo** - Funciona perfeitamente em desktop, tablet e mobile
- **Animações Suaves** - Efeitos visuais elegantes ao rolar a página
- **Seções Completas**:
  - Home com hero section impactante
  - Sobre a empresa com histórico e estatísticas
  - Serviços de logística esportiva
  - Competições atendidas (Brasileirão, Copa do Brasil, etc.)
  - Galeria audiovisual
  - Formulário de contato funcional

## 🎨 Seções do Site

### 1. Home
- Apresentação da marca
- Destaque para 8 anos de experiência
- Call-to-action para serviços e contato

### 2. Sobre
- História da empresa (desde 2017)
- Marco inicial: 05/06/2017 - Atlético Goianiense
- Estatísticas: 8+ anos, 50+ clubes, 100+ eventos
- Equipe especializada em esporte

### 3. Serviços
- Transporte de Delegações
- Transporte de Equipamentos
- Hospedagem e Alimentação
- Coordenação de Eventos
- Logística Aérea
- Suporte Operacional

### 4. Clientes/Competições
- Brasileirão (Séries A, B, C e D)
- Copa do Brasil
- Copa do Nordeste
- Futebol Feminino
- Categorias de Base
- Campeonatos Estaduais

### 5. Galeria Audiovisual
- Grid moderno com momentos marcantes
- Placeholders para fotos e vídeos reais

### 6. Contato
- Formulário completo
- Links para Instagram (@twister.srv) e WhatsApp
- Informações de localização (Bahia, Brasil)

## 🚀 Como Visualizar

O servidor já está rodando! Acesse:
- **URL**: http://localhost:8080

Ou abra o arquivo `index.html` diretamente no seu navegador.

## 🎯 Próximos Passos (Personalização)

Para completar o site com conteúdo real:

1. **Imagens Reais**
   - Substitua os gradientes das galerias por fotos reais dos eventos
   - Adicione fotos de jogos, transportes, equipes

2. **Vídeos**
   - Incorpore vídeos reais na seção audiovisual
   - Use YouTube, Vimeo ou arquivos locais

3. **Dados de Contato**
   - Atualize email real
   - Configure número de WhatsApp correto
   - Adicione mais redes sociais se necessário

4. **Depoimentos**
   - Adicione seção com depoimentos de clientes
   - Logos de clubes atendidos

5. **SEO**
   - Otimize meta tags para busca
   - Adicione sitemap.xml
